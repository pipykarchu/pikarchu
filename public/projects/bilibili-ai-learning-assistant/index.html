PRD 项目 · Web 学习工具

B站视频 AI 学习助手

把 B 站视频片段转成逐字稿、人物区分、学习纲要、脑图和 PPT 大纲，降低长视频学习和复习成本。

核心流程

1
输入视频

支持 B 站链接和时间区间；不能绕过权限，不读取 Cookie。

2
转写与说话人区分

输出原始逐字稿、整理稿、时间戳和人物标签。

3
学习结构化

生成学习纲要、重点回看位置、Mermaid 脑图和 PPT 大纲。

4
本地保存与导出

历史结果默认存在浏览器 IndexedDB，支持 Markdown 导出。

MVP 边界

本项目当前作为 PRD 和产品方案展示，重点说明学习场景、合规边界、任务状态、本地历史、AI 成本和导出结果，不做会员/付费视频绕过。

视频学习 ASR Diarization Markdown Mermaid PPT 大纲
成功标志：用户输入一个可访问视频后，能拿到可复习、可跳转、可导出的学习材料。
