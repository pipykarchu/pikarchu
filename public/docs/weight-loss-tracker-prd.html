1. 背景与目标

减肥记录不只看体重，还要结合围度、目标差距、运动强度、饮食模式和生理期状态。这个工具用单文件 HTML 保留本地数据，方便持续记录和复盘。

目标成功标志
低成本记录打开 HTML 即可新增记录，不依赖账号
周期友好建议经前、经期、经后和排卵期建议不同
可迁移数据支持 JSON 导入导出

4. 使用流程

  • 打开减肥记录仪 HTML。
  • 填写当天体重、围度、目标、运动和饮食状态。
  • 选择生理期状态和身体感受。
  • 保存记录后查看指标卡、今日建议、趋势图和历史表。
  • 需要迁移时导出 JSON，需要恢复时导入 JSON。
