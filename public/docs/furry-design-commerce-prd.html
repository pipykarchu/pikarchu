2. MVP 范围

模块要做什么不做什么
套餐设计头像尝鲜包、表情包小套装、基础兽设包、商用升级包不承诺复杂动画或大型 IP 企划
需求收集收集风格、角色性格、用途、授权范围和参考偏好不收集敏感信息
视觉产出AI 生成候选图，人工筛选和轻修不模仿知名 IP、不复刻真人
交付包装输出图片、授权说明、使用建议和修改规则不做无限次改稿

3. 套餐草案

套餐内容适用场景
头像尝鲜包1 个角色方向、若干候选头像、1 次轻修改社交头像、试水需求
表情包小套装固定角色 + 3-6 个表情动作私域聊天、小红书/B站互动
基础兽设包角色三视图/设定图、性格关键词、基础授权说明OC 设定、长期内容创作
商用升级包更清晰授权边界、交付规格、使用场景说明店铺物料、活动视觉、轻商用

4. 核心流程

  • 用户提交角色类型、风格、用途、授权范围、预算和交付时间。
  • 先明确不做侵权 IP、明星脸、真人复刻和模糊商用授权。
  • 生成多个候选方向，人工筛选可用结果。
  • 用户确认方向并标注喜欢点和修改点。
  • 交付最终图、授权说明、使用建议和文件命名。
  • 复盘咨询来源、成交率、改稿次数、交付时长和复购意向。

5. 风险与边界

风险对策
侵权或仿 IP禁止模仿知名 IP、明星脸、真实人物肖像
商用边界不清每单交付授权说明，区分自用、平台发布、商用
AI 图质量不稳定候选图先筛选，复杂需求不承诺一次到位
改稿失控套餐内写清修改次数和超出后的处理方式
