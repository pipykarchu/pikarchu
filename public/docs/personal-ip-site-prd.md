# 皮玺玉个人 IP 站 PRD 摘要

## 当前定位

面试演示作品集版。目标是在 1 分钟内让访客看懂：皮玺玉是谁、做过什么 AI 项目、如何联系、这个站点如何体现产品和工程能力。

## MVP 范围

- 首屏个人定位
- 作品分区与作品卡片
- 项目详情弹窗
- 服务与联系入口
- 智能客服
- 留言保存
- 行为埋点
- 运营后台
- 面试演示包
- 隐私与数据用途说明

## 面试演示路径

1. 先讲个人定位：产品经理 × AI 实践者，把痛点做成可复用工作流。
2. 主讲两个项目：求职作战中心、提单履行预警看板。
3. 展示方法复用：PRD Writer Skill、漫剧生产工作流 Skill、剧本和分镜 Skill。
4. 跑通闭环：首页、作品详情、Demo、智能客服、留言、后台。
5. 说明边界：当前是作品集和面试演示版，小程序商业化后续独立立项。

## 上线前风险

- 公开上线前必须替换默认后台账号、密码和密钥。
- API Key 只能放在后端环境变量，不能进入前端代码。
- 留言和埋点会记录访客主动提交的信息和基础行为，需要保留隐私说明。
- 简历 PDF、真实二维码、真实运营数据和线上域名仍需补齐。
