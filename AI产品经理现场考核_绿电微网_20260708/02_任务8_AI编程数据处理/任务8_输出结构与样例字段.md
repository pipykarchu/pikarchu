# 任务8：AI编程实现电力数据轻量化处理

## 推荐选题

选择：招标数据自动去重、分类、标签筛选。

原因：最贴近绿电微网公司的招投标业务，也最容易在 30-40 分钟内跑出可见结果。

## 成果文件

1. `sample_tender_data.csv`：样例招标数据
2. `tender_processor.py`：处理脚本
3. `processed_tender_data.csv`：处理结果
4. `任务8_招标数据处理说明.md`：设计思路与业务落地说明
5. 代码运行截图

## 输入字段

| 字段 | 说明 |
| --- | --- |
| project_name | 项目名称 |
| buyer | 采购单位 |
| publish_date | 发布时间 |
| notice_type | 公告类型 |
| link | 公告链接 |

## 输出字段

| 字段 | 说明 |
| --- | --- |
| project_name | 去重后的项目名称 |
| buyer | 采购单位 |
| publish_date | 发布时间 |
| notice_type | 公告类型 |
| link | 公告链接 |
| description | 项目需求描述 |
| project_type | 自动分类结果 |
| tags | 关键词标签 |
| priority | 跟进优先级 |
| reason | 分类原因 |

## 分类规则

| 标签 | 关键词 |
| --- | --- |
| 直流微网 | 直流、微网、微电网、柔性直流、直流母线 |
| 充电桩 | 充电桩、充换电、充电站、有序充电 |
| 储能 | 储能、电池、削峰填谷、储能系统 |
| 光伏 | 光伏、分布式光伏、太阳能、屋顶光伏 |
| 数据中心供电 | 数据中心、算力中心、机房、UPS、供电系统 |
| 园区电力改造 | 园区、配电改造、节能改造、能源管理 |

## 优先级规则

- 高：同时命中两个以上核心标签，或包含数据中心、直流微网、光储充、零碳园区
- 中：命中一个核心标签，且项目描述较明确
- 低：只命中泛能源词，需人工复核

## 业务落地说明

这个小工具可以作为招投标线索筛选的第一步，把人工每天翻公告的工作变成“机器先筛、产品/销售再复核”。后续可接入南方电网采购公告、企业微信推送、CRM 线索池和智能体方案生成。
